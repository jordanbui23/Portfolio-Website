Project information

  • Project: Map Viewer
  • Category: Data Structures and Algorithms | Quad Trees
  • Language/Libraries/APIs: C#
  • Project date: 26 October, 2021
  • Project URL: Get the .exe file to work to demo

Description

  • Developed a program that created a map from a .csv file containing floating point numbers that give the:
    1. The (floating-point) x-coordinate of one end of the line segment.
    2. The (floating-point) y-coordinate of this end of the line segment.
    3. The (floating-point) x-coordinate of the other end of the line segment.
    4. The (floating-point) y-coordinate of this end of the line segment.
    5. An integer giving the color of the line segment in ARGB format.
    6. A floating-point number giving the line width in pixels.
    7. An integer giving the lowest zoom level at which this segment should be shown.
  • Used GitHub for Source Control and Visual Studio for development and debugging
  • The data structure uses for storing the map data is known as a quad tree. The root of the quad tree represents the entire map.The map is divided into four equal-sized quadrants by splitting in half vertically, then splitting these halves in half horizontally. The root node then has four children - one for each of these quadrants. These nodes each have four children, and so on, until the map is described at the finest detail needed.
  • The zoom levels range from a minimum (1) that can display several towns at once, to a maximum (7) capable of detailing the lanes in a parking lot.
